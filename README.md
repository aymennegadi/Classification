# Chapter 1: Introduction to Classification (LaTeX Source & PDF)

Welcome to the repository for **Chapter 1 — Introduction to Classification**. This repository contains the complete LaTeX source code, structural assets, and documentation for a comprehensive introductory guide to Machine Learning and Classification paradigms.

---

## Chapter Overview

This chapter serves as a foundational guide to classification in Machine Learning. It bridges the gap between basic ML theory and concrete mathematical evaluations used in real-world pipelines.

### Key Topics Covered:
*   **Introduction to ML:** AI vs. ML, real-life examples, and the 3 learning paradigms (Supervised, Unsupervised, Reinforcement).
*   **Supervised Learning:** Deep dive into Features ($X$), Target Variables ($y$), and training/testing splits.
*   **Classification vs. Regression:** Conceptual differences and clear comparison tables.
*   **Classification Workflow:** The step-by-step pipeline from data collection to model deployment and improvement.
*   **Evaluation Metrics:** In-depth breakdown of the **Confusion Matrix**, **Accuracy**, **Precision**, **Recall**, and **F1-Score** with complete mathematical derivations and scenarios on when to use which.
*   **Algorithms Overview:** High-level summary of the 7 classic classification algorithms (Logistic Regression, Decision Trees, KNN, Naive Bayes, SVM, Random Forest, and GBM).
